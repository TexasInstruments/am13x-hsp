Installation:

Download and install the pre-requisites:

Code Composer Studio - Download and install CCS 20.5.0

  • Obtain CCS_20.5.0.00028_win.zip from https://www.ti.com/tool/download/CCSTUDIO
  • Extract the zip and run the executable ccs_setup_20.5.0.00028.exe
  • Use Installation Directory as C:\ti\ccs2050
  • in Select Components, Choose, 'AM13x Arm-based microcontrollers'
  • Wait for completion of the installation (approx. 10 mins) and click 'Finish'

AM13E2X SDK - Download and install the version mentioned above

  • Obtain am13e230x_sdk_26_00_00_06_STS-windows-x64-installer.exe from AM13E2X-SDK Software development kit (SDK) | TI.com
  • Run the executable
  • Use Installation Directory as C:\ti
  • Wait for completion of the installation (approx. 5 mins) and click 'Finish'

TI System Configuration Tool

  • Obtain sysconfig-1.27.0_4565-setup.exe from from SYSCONFIG IDE, configuration, compiler or debugger | TI.com
  • Run the executable
  • Use Installation Directory as C:\ti
  • Wait for completion of the installation and click 'Finish'

MATLAB/Simulink

  • Download and install R2025b MATLAB & Simulink with the following products: MATLAB Coder, Simulink Coder, Embedded Coder
  • Motor Control Blockset for using motor control example
  • Simscape, Simscape Electrical for high fidelity simulation
  • Embedded Coder Support Package for ARM Cortex-M Support Processors (Base support package to use this AM13x specific support)
  • MinGW is an important compiler used as MEX compiler by many MATLAB workflows and also for make utility.

MinGW-w64 Compiler

MinGW is an important compiler used as MEX compiler by many MATLAB workflows and also for the make utility.
To install the MinGW-w64 compiler:
  • On the MATLAB Home tab, in the Environment section, click Add-Ons
  • In the Add-Ons Explorer, search for MinGW
  • Select MATLAB Support for MinGW-w64 C/C++/Fortran Compiler and install
Important: MinGW Installation Folder Cannot Contain Spaces
Do not install MinGW in a location with spaces in the path name.
  • Incorrect: C:\Program Files\mingw-64
  • Correct: C:\mingw-64
Installing MinGW in a path with spaces will cause build errors and prevent proper compilation of MEX files and generated code.
