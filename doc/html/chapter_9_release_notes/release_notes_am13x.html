Release Notes - Version 26.0.0

Features
Core Capabilities:
  • TI ARM Clang and IAR toolchain support
  • SysConfig integration for hardware initialization
  • CCS IDE project generation
  • Processor-in-the-Loop (PIL) with profiling
  • External Mode for real-time parameter tuning (XCP over Serial)
Code Optimization:
  • TMU Code Replacement Library for trigonometric functions
Hardware Support:
  • AM13E230x LaunchPad
Blocks:
  • Hardware Interrupt block
Known Limitations
  • Peripheral blocks not available in this release (planned for future)
Requirements
  • MATLAB R2025b or higher
  • CCS 20.5.0
  • AM13E2X SDK 26.00.00.06
  • SysConfig 1.27.0
Examples
  • Led blink, PIL, External mode and Motor control examples
