Download the HSP Package

Method 1: Download via MATLAB Add-On Explorer

Step 1. Open MATLAB
Step 2. Click Add-Ons → Get Hardware Support Packages
Step 3. In the Add-On Explorer window, use the 'Search for add-ons' field to search for 'AM13x'
Step 4. From the dropdown, choose 'Embedded Coder Support Hardware Support Package for TI AM13x'
Step 5. The add-on page will open
Step 6. Click Add → Download Only
Important: Do NOT use Add → Add to MATLAB (Download and add to ath)
Step 7. Sign in using your MathWorks Account when prompted
Step 8. Download the zip file

Method 2: Direct Download from MathWorks File Exchange

Step 1. In your browser, navigate to the MathWorks File Exchange page for AM13x HSP:
https://www.mathworks.com/matlabcentral/fileexchange/[package-id]-embedded-coder-hardware-support-package-for-ti-am13x
Step 2. Click Download
Step 3. Sign in using your MathWorks Account when prompted
Step 4. Download the zip file

Method 3: Download from Github

Step 1. Search am13x-hsp on github
Step 2. Click on the repository under Texas Instruments' official account
Step 3. Go to Releases and download the zip file
Step 3. Alternatively, clone the repo, or download the zip of the main branch
Note: depending on the way you download/clone, the download name/ default name of the folder after you extract will vary. For consistency, we are using am13x-hsp-xxxxxxx for the downloaded/extracted folder name

Install the HSP package:

Step 1. Download the support package from any one of the methods mentioned above
Step 2. Open MATLAB
Step 3. Ensure to set 'Settings'->'MATLAB'->'Add-Ons'->'Installation Folder' to a path with no spaces. For example: C:\workarea\AddOns
Step 4. Extract the zip file to the installation folder. For example: C:\workarea\AddOns\am13x-hsp-xxxxxxx
Step 5. In the MATLAB environment, browse to the installed folder. For example: C:\workarea\AddOns\am13x-hsp-xxxxxxx folder
Step 6. Run command hsp_am13x_setup in the Command Window
Step 7. Expect a dialog box which prompts for the paths of CCS, SDK, compiler and SysConfig. Click OK
Note: The paths used, will be stored in hsp_am13x_config.json file for future setup as default paths. User can also use the CLI version of setup, to avoid the pop up path input window, and instead pass the paths as named parameter. No parameters will result in the use of default paths. For more information, check description in hsp_am13x_setup_cli.m file.
Step 8. Expect following logs in the Command Window

Troubleshooting:

  1. If installation path contain spaces, some of the tools used for building and deployment do not work.
  2. If you are installing using .mltbx, change the default paths of Add-Ons
